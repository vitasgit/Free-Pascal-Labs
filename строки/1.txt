В Free Pascal работа со строками имеет несколько ключевых моментов:

1. Объявление строк:
```pascal
var
  s: string; // динамическая строка
  s2: string[20]; // строка фиксированной длины до 20 символов
```

2. Основные операции:
```pascal
s := 'Привет'; // присваивание
Length(s); // длина строки
s + s2; // конкатенация
s[1]; // первый символ
Delete(s, 2, 3); // удаление подстроки
Insert('текст', s, позиция);
```

3. Полезные функции:
```pascal
UpperCase(s); // в верхний регистр
LowerCase(s); // в нижний регистр
Trim(s); // удаление пробелов
```

4. Преобразования:
```pascal
IntToStr(число); // число в строку
StrToInt(строка); // строка в число
```
