Я делаю учебный проект на тему "создания векторных изображений"
Я использую Free Pascal (FPC).
Суть проекта: разработать "учебное" подобие paint приложения.

Основные компоненты, используемые в проекте:
TPaintBox - для визуализации набора векторных примитивов и организации интерактивного управления ими.
TFlowPanel с набором TSpeedButton - для формирования панели управления

Вспомогательные компоненты, используемые в примере:
TMainMenu - для формирования главного меню
TStatusBar - для визуализации текущего состояния приложения
TPanel - для группировки компонентов

В приложении должно быть реализовано следующие:
На панели управления 5 кнопок:
1) Линия. Для рисовании линии. (простая линия)
2) Квадрат. Для рисования произвольного квадрата (прямоугольника).
3) Элипс. Для рисования произвольного элипса.
4) кнопка перемещения. Перемещает выбранные объекты на холсте
5) кнопка удаления. Удаляет выбранный объект на холсте
