## **TControl.OnMouseUp**

**Обработчик события**, вызываемый при обработке события отпускания кнопки мыши (mouse up) для элемента управления.

### **Объявление**

Исходная позиция: `controls.pp`, строка 1591

```
protected property TControl.OnMouseUp : TMouseEvent
  read FOnMouseUp
  write FOnMouseUp;
```

### **Описание**

Событие `OnMouseUp` вызывается (если назначен обработчик) из метода `MouseUp`. Оно срабатывает, когда стандартные события мыши включены через свойство `ControlStyle`.

Событие вызывается **после обновления позиции мыши** и после того, как событие обработано активным `DragManager`.

Приложение должно реализовать и назначить обработчик типа `TMouseEvent`, который будет реагировать на это событие.

Используй:

* `OnMouseDown` — для действий при нажатии кнопки мыши
* `OnMouseMove` — для действий при перемещении указателя мыши
* `OnMouseWheel`, `OnMouseWheelDown`, `OnMouseWheelUp` — для обработки прокрутки колеса мыши

## **TControl.OnMouseDown**

**Обработчик события**, вызываемый при нажатии кнопки мыши (mouse down) на элементе управления.

### **Объявление**

Исходная позиция: `controls.pp`, строка 1589

```
protected property TControl.OnMouseDown : TMouseEvent
  read FOnMouseDown
  write FOnMouseDown;
```

### **Описание**

Событие `OnMouseDown` вызывается (если назначено) из метода `MouseDown`.

Оно происходит:

* после того, как родительская форма получает фокус
* после отмены активного элемента редактирования (например, `Edit`)

Если активен `DragManager`, он уже обновил позицию мыши до вызова события.

Приложение должно реализовать и назначить обработчик типа `TMouseEvent`.

Используй:

* `OnMouseUp` — для обработки отпускания кнопки мыши
* `OnMouseMove` — для обработки перемещения мыши
* `OnMouseWheel`, `OnMouseWheelDown`, `OnMouseWheelUp` — для обработки прокрутки

## **TControl.OnMouseMove**

**Обработчик события**, вызываемый при перемещении указателя мыши внутри элемента управления.

### **Объявление**

Исходная позиция: `controls.pp`, строка 1590

```
protected property TControl.OnMouseMove : TMouseMoveEvent
  read FOnMouseMove
  write FOnMouseMove;
```

### **Описание**

Свойство `OnMouseMove` — это обработчик типа `TMouseMoveEvent`, который вызывается при обработке сообщения `LM_MOUSEMOVE`.

Событие:

* вызывается из метода `MouseMove`
* происходит **после обновления позиции мыши**
* после уведомления `DragManager` о новой позиции указателя

Приложение может реализовать обработчик (метод объекта), чтобы выполнять действия при изменении положения мыши внутри элемента.

В обработчик передаётся:

* `Sender` — элемент управления, в котором произошло событие

Используй:

* `OnMouseDown` и `OnMouseUp` — для обработки нажатия/отпускания кнопок мыши
* `OnMouseEnter` и `OnMouseLeave` — для отслеживания входа/выхода курсора из элемента
* `OnMouseWheel` — для обработки прокрутки колесом мыши
